# selfbot-icon
Este projeto em Node.js utiliza a biblioteca discord.js-selfbot-v13 para fins de estudo de APIs. O script conecta-se a uma conta de usuário, acessa o histórico de um canal específico, filtra anexos do tipo imagem (PNG, JPG, GIF) e os encaminha automaticamente para uma DM destino. Inclui sistema de delay para evitar limites de taxa da plataforma.
